Использование утилиты Text Tools                                              
Для отображения диалогового окна утилиты Text Tools щелкните на кнопке Text Tools, находящейся в группе Utilities на вкладке Главная.                                    
Утилита Text Tools всегда работает с текущим выделенным диапазоном. Диалоговое окно Text Tools всегда  "остается сверху". Это означает, что облегчается выбор различных диапазонов ячеек, а также их обработка Может выделятьс яотдельный диапазон ячеек, строки или столбцы, несколько выделенных областей либо весь лист в целом.    
По завершению работы с  Text Tools щелкните на кнопке Закрыть.                                              
                                                         
                                                         
Изменение регистра символов текста                                                  
1.В диалоговом окне Text Tools выберите операцию Изменить регистр.                                          
2. Выберите соответствующий параметр Изменить для преобразования выделенного текста.                                    
3. Щелкните на кнопке Применить для изменения выделенных ячеек.                                          
                                                         
                                                         
Добавление текста                                                      
1. Выделите изменяемые ячейки.                                                  
2. В диалоговом окне Text Tools выберите подходящую операцию добавления текста.                                      
3. Введите текст, который нужно добавить в поле Добавляемый текст.                                        
4. В раскрывающемся списке Позиция символа выберите параметр, который соответствует позиции, в которой будет добавлен текст.                            
5. Если нужно ограничить операции выделенной ячейкой, содержащей текст, выберите параметр Пропустить не текстовые ячейки. Если этот параметр не выбран, будут изменяться также ячейки, содержащие значения.        
6. Щелкните на кнопке Применить для изменения выделенных ячеек.                                          
                                                         
                                                         
Удаление текста в выбранной позиции                                                  
1. Выделите ячейки, которые нужно изменить.                                              
2. В диалоговом окне Text Tools выберите параметр Удалить в позиции.                                        
3. В раскрывающемся списке Удаляемые символы выберите количество удаляемых символов.                                    
4. В раскрывающемся списке Позиция символа выберите позицию, начиная с которой будут удаляться символы.                                
5. Если нужно ограничить операцию выделенными ячейками, которые включают текст, выберите параметр  Пропустить нетекстовые ячейки. Если этот параметр не выбран, также изменяются ячейки, содержащие значения.        
6.Щелкните на кнопке Применить для изменения выделенных ячеек.                                          
                                                         
                                                         
Удаление пробелов                                                    
1. Выберите изменяемые ячейки.                                                  
2. В диалоговом окне Text Tools выберите параметр Удалить пробелы.                                          
3. В раскрывающемся списке Удалить выберите параметр, который удаляет нужные пробелы.                                    
4. Щелкните на кнопке Применить для изменения выделенных ячеек.                                          
Примечание. Результат выбора параметра Удалить все лишние пробелы эквивалентен выбору функции Excel СЖПРОБЕЛЫ.                                      
                                                         
                                                         
Удаление символов по типу                                                    
1. Выделите изменяемые ячейки                                                  
2. В диалоговом окне Text Tools выберите параметр Удалить символы.                                          
3. В раскрывающемся списке Удалить все выберите параметр, соответствующий типу удаляемых символов.                                
4. Щелкните на кнопке Применить для изменения выделенных ячеек.                                          
Примечание. Выбор параметра Не печатаемые символы эквивалентен выбору функции Excel ПЕЧСИМВ.
