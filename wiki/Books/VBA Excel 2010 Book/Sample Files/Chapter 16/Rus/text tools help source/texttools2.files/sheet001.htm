Советы по работе с утилитой Text Tools                                                                
Диалоговое окно утилиты Text Tools остается открытым после щелчка на кнопке Применить, поэтому можно вносить другие изменения в выделенные ячейки. Либо можно выбрать другой диапазон ячеек.                              
После щелчка на кнопке Применить можно щелкнуть на кнопке Отменить  (либо нажать клавиши Ctrl+Z) для отмены выбранного эффекта. Доступен только один уровень отмены.                                    
Утилита Text Tools работает только с непустыми ячейками. Таким образом можно выделять целые строки или столбцы. Обратите внимание, что утилиту Text Tools невозможно использовать для добавления текста в пустые ячейки.                        
Если установлен флажок Пропускать пустые ячейки, утилита будет работать только с ячейками в выделенной области, которые содержат текст (а не значения). Если этот флажок не устанавливать, утилита будет работать с ячейками, которые включают текст либо значения. Пустые ячейки либо ячейки с формулами всегда пропускаются.
При удалении пробелов из текста выбор параметра Все избыточные пробелы приведет к преобразованию нескольких пробелов в один пробел, а такжу к удалению всех пробелов в начале и в конце текста.                              
Утилита Text Tools может применяться только к тексту в ячейках рабочего листа. Например, утилиту невомзможно использовать для изменения регистра надписей диаграммы.                                    
Утилита Text Tools работает только с текстом, введенным непосредственно в ячейки, а формулы игнорирует. Например, применением утилиты Text Tools к ячейке, содержащей такую формулу, как  =ПРОПИСН(A1), даже если эта формула отображает текст в ячейке, завершится безрезультатно.
